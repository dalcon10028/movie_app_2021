## 🎥 클론 코딩 영화 평점 웹서비스

### 시작하기

```bash
npm i # 의존성 설치
npm start # development mode로 시작
```

### 테스트

```bash
npm run test # watch mode로 테스트 실행
```

### 빌드

```bash
npm run build # build폴더로 빌드된 파일 생성
```
